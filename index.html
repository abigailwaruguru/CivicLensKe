About CivicLens

CivicLens is a web application designed to promote transparency and accountability in campaign finance. It allows users to securely register, log in, and record political donations. All donations are automatically checked for compliance, and users can download reports for regulatory purposes.

How It Works

  1. Sign Up: Create a secure account using the Sign Up page.
  2. Log In: Access your account from any device using your credentials.
  3. Enter Donations: Use the donation portal to record new donations. Each entry is checked for compliance and flagged if suspicious or in violation.
  4. View Dashboard: See all your donation records and download regulator-ready reports in CSV or JSON format.
  5. Log Out: Securely log out at any time using the Logout button in the portal.
